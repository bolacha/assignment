# assignment
Assignment for Interview - Front End Skills

My configuration for running this code :

- NodeJS 8.9.0
- Yarn

First ,

``` yarn install ```

``` yarn start ```

This will open a browser , if that did not happen please try the URL : http://localhost:3000/
